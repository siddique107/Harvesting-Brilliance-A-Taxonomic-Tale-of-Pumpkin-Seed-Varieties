# Harvesting Brilliance: A Taxonomic Tale of Pumpkin Seed Varieties

## Project Overview
**Harvesting Brilliance** is a machine learning based web application that classifies pumpkin seed varieties using morphological measurements.  

The system takes numerical features such as Area, Perimeter, Major Axis Length, Minor Axis Length, and other shape-based properties as input and predicts the **category of pumpkin seed variety**.

This project helps in automated seed classification and supports agricultural research using intelligent prediction systems.

---

## Features
- User-friendly web interface for inputting seed measurements
- Machine Learning model for accurate seed variety classification
- Instant prediction of pumpkin seed category
- Clean and responsive UI

---

## Technologies Used
- Python
- Machine Learning
- OpenCV
- Scikit-learn

---

## How to Run

1. Install requirements:
```bash
   pip install -r requirements.txt
```

2. Run project:
```bash
   python app.py
```
